# Personal Portfolio Website with Modern UI and Animations

A visually stunning, high-performance personal portfolio website built with React, Tailwind CSS v4, and Framer Motion.

## 🛠️ Tech Stack
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Smooth Scroll**: [React Scroll](https://www.npmjs.com/package/react-scroll)

## ✨ Features
- **Modern UI**: Dark mode first design with glassmorphism and vibrant gradients.
- **Advanced Animations**:
  - Parallax scrolling effects on hero background.
  - On-scroll reveals for all major sections.
  - Staggered list animations for skills and projects.
- **Fully Responsive**: Optimized for Mobile (375px), Tablet (768px), and Desktop (1280px).
- **Accessibility**: Support for `prefers-reduced-motion`.
- **Performance**: High Lighthouse scores and optimized assets.

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/DileepKumarReddySathi/Personal-Portfolio-Website-with-Modern-UI-and-Animations.git
   cd Personal-Portfolio-Website-with-Modern-UI-and-Animations
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
